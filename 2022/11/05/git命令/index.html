1.3git基本命令

序号 功能 命令
1 拉取远程代码 git git pull
2 查看本地分支 git branch
3 查看远程分支 git branch -r
4 查看所有分支 git branch -a
5 创建本地分支 git branch branchName
6 建立本地分支与远程分支的联系 git branch –set-upstream-to=origin/< branch > feture-test
7 重命名分支 git branch -m old new / git branch -M old new
8 删除本地分支 git branch -a
9 删除远程分支 git branch -d -r branchname
10 查看在你上次提交之后是否有对文件进行再次修改 git status
11 删除远程分支 git add .
12 删除远程分支 git commit -m
13 将远程仓库origin的master分支与本地仓库master分支关联 git push -u origin master
13 提交代码 git push

2.git配置忽略文件

以斜杠“/”开头表示目录；
以星号“*”通配多个字符；
以问号“?”通配单个字符；
以方括号“[]”包含单个字符的匹配列表；
以叹号“!”表示不忽略(跟踪)匹配到的文件或目录
